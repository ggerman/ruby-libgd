# GD::Gif.new

```ruby
gif = GD::Gif.new("output.gif", loop: true)
```

新しいアニメーションGIFライターを作成します。

---

## 概要

`GD::Gif.new` は、アニメーションGIFの生成を開始するための
GIFエンコーダを初期化します。

この時点では GIF ファイルは作成されますが、
フレームが追加されるまで内容は書き込まれません。

---

## 引数

### `path` (`String`)

出力する GIF ファイルのパスを指定します。

```ruby
GD::Gif.new("animation.gif")
```

---

### `loop` (`Boolean`, 任意)

アニメーションの再生ループを制御します。

- `true`（デフォルト）  
  アニメーションは無限にループ再生されます。

- `false`  
  アニメーションは1回のみ再生されます。

```ruby
GD::Gif.new("once.gif", loop: false)
```

---

## 挙動

- 初期化時にファイルが作成されます
- 最初の `add_frame` 呼び出し時に GIF ヘッダが書き込まれます
- フレームが1つも追加されなかった場合、空のファイルになります
- 実際のエンコード処理は `add_frame` で行われます

---

## 注意事項

- GIF ファイルは `close` を呼び出すことで正しく終了します
- `close` を呼び出さずに終了した場合、GIF が破損する可能性があります
- 同じインスタンスで複数の GIF を作成することはできません
